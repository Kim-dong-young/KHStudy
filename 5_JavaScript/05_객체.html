객체

* 객체 선언 및 사용

객체는 중괄호 {}를 사용해서 생성하고
중괄호 안에 이 객체에서 필요한 속성들을 정의함
속성명 : 속성값 ( key : value )형태로 정의한다.
속성값으로는 모든 자료형의 값을 담을 수 있음 

        변수 = {
            속성명 : 속성값(함수도 가능),
            속성명 : 속성값,
            속성명 : 속성값,
            ...
        }

* 객체의 메소드 속성

객체의 속성 중 함수자료형인 속성을 메소드라고 부름
