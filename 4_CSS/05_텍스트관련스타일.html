텍스트 관련 스타일(text~)

color : 텍스트 색상 지정

            선택자{
                color:색상명 | #16진수(코드값) | rgb(x,x,x) | rgba(x,x,x,x)
            }

text-decoration : 텍스트에 줄을 긋거나 줄을 없앨 때 사용

            선택자{
                text-decoration : none | underline | overline | line-through;
            }

text-align : 텍스트 정렬할 때 사용

            선택자{
                text-align:left(기본값) | right | center | justify
            }

왼쪽정렬(기본값)

(엑스포츠뉴스 김근한 기자) 이 정도면 '푸른 한화' 신드롬이 아닐까.
전통적인 주황색이 아닌 푸른색을 입은 한화 이글스가 상대 팀들에 공포의 존재로 변신했다.
한화가 푸른 원정 유니폼을 입고 거둔 성적은 무려 14승 3패다.
이제 한화는 5강 경쟁자 가운데 가장 많은 잔여 맞대결(8차례)을 치르는 롯데 자이언츠 원정으로 향한다.
주축 타자인 채은성과 안치홍도 복귀할 예정이라 더 두려울 게 없다.
