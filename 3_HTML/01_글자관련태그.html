문단을 나누는 태그 : p, pre

문단 영역을 나누는 태그로는 p 태그와 pre 태그가 있다.
p 태그는 줄바꿈을 하고자한다면 별도의 태그를 작성해야한다.
공백은 한 개의 공백만을 표시하기 때문에
별도의                기호문구(nbsp;)를 기술해야 함 

            pre태그는 시작태그에서부터 종료태그까지 입력된 내용을 그대로 표현하는 태그
            줄바꿈이라던가              여러개의 공백을 그대로 인식함(Tap 들여쓰기 포함)

그 밖에 글자를 다루는 태그글

일반글꼴(태그로 감싸지 않은 텍스트)

b : 글자를 굵게 표시하는 태그

strong : 글자를 굵게 표시하는 태그 + 스크린리더

em : 글자를 기울여서 보여주는 태그

i : 글자를 기울여서 보여주는 태그

mark : 형광펜 효과를 주는 태그

u : 글자에 밑줄이 그려지는 태그

s : 글자에 취소선을 넣어주는 태그

small : 글자를 작게 표현해주는 태그

기본글자에 sub : 아래첨자를 나타내는 태그 sup : 윗첨자를 나타내는 태그

태그를 중첩해서 사용하면 가능한가?
